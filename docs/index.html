Feline Frenzy

Help our sweet old granny find her cat, Lua.
Watch out! Enemies and obstacles await along the way...

A small pixel art game, built with Lua and run on PICO-8 — a retro-style virtual console.

Everything was created directly within PICO-8, using its built-in tools:

  • Code editor (Lua)
  • Sprite editor
  • Sound and music editor
  • Map and tile editor
